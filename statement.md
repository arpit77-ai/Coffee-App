## Problem Statement

Customers need a simple way to browse a coffee menu, select items, and place an order without confusion. Traditional manual ordering can be slow or error-prone, so a streamlined digital console-based solution is required for quick, accurate ordering.


## Scope of the Project

* Develop a Python console application for ordering coffee.
* Display menu items with prices.
* Allow users to add items to an order.
* Provide order review, total calculation, and checkout.
* Focus on core functionality, not graphical interfaces or payment processing.


##  Target Users

* Beginners learning Python and OOP concepts
* Students practicing console applications
* Small caf√© prototypes or demo systems
* Anyone wanting a simple order-simulation program


## High-Level Features

* View a menu of coffee items
* Add items to an order
* View all items in the current order
* Automatic total price calculation
* Checkout confirmation
* Clean, interactive console UI
